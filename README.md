# Zakat Calculator App

## Description
This is a simple Android application that calculates gold zakat.  
Users can enter gold weight, gold price and select type (keep or wear) to calculate zakat amount.

## Features
- Calculate gold zakat
- Input validation
- Share app feature
- About page
